# Basic k8s

## Delete service
```js
kubectl delete svc service_name
```

#### Get all service
```js
kubectl get service
```

#### Create service
```js
kubectl create -f service.yaml 
```

#### Get all pods
```js
kubectl get pods
```

#### Create deployment
```js
kubectl create -f deployment.yaml
```

#### Delete deployment
```js
kubectl delete deployment nestjs-k8s-dungtv
```
